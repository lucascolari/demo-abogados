# Estudio Jurídico Demo

Demo de landing page para estudios jurídicos, con **asistente legal** (chat) que
orienta consultas y agenda turnos. Sitio de demostración usado como herramienta de
venta para captar clientes del rubro legal.

## Stack
- HTML + CSS + JavaScript **vanilla** (sin dependencias ni build).
- Se abre con doble clic en `index.html`.

## Estructura
- `index.html` — estructura de la página
- `styles.css` — diseño (paleta azul marino + dorado)
- `script.js` — interacciones + asistente legal simulado

> El asistente es una simulación (respuestas por palabras clave). No usa IA real
> ni envía datos a ningún servidor.
